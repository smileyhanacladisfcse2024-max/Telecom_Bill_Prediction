# Telecom Bill Prediction System

## Project Overview
Machine Learning system to predict telecom customers' next monthly bill and identify customers with significant bill drops.

## Features
- Next Month Bill Prediction
- Bill Drop Detection
- Personalized Offer Recommendation
- Automated Email Generation
- Streamlit Dashboard
- Customer Scorecard Export

## Tech Stack
- Python
- Pandas
- Scikit-Learn
- Streamlit

## Dataset
IBM Telco Customer Churn Dataset

## Outputs
- Predicted bills
- Bill-drop customers
- Offer recommendations
- Email templates
- Dashboard
- CSV export

python src/final_pipeline.py
